# PRODIGY_WD_01

- Modern Navigation Menu

=Overview
.This project features a modern and responsive navigation menu designed for seamless user experience across different devices. The navigation bar includes a standard menu with links, a mobile-friendly hamburger menu, and an additional three-dots dropdown for extra options.

=Features
.Responsive navigation menu with a mobile-friendly toggle.
.Smooth color transitions on scroll.
.Dropdown menu for extra navigation options.
.Simple and clean HTML, CSS, and JavaScript structure.
